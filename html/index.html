MAL2Go: The MyAnimeList API Wrapper For GoLang

"That time I created an API wrapper for MyAnimeList..."

MAL2Go is an easy-to-use MyAnimeList API wrapper written in Go. It can be used to access the official MyAnimeList V2 API. Currently it is mature enough to safely get and set information about an anime/the authenticated user's anime list.

It has complete functionality to do the same with the manga section of MAL, but it hasn't been tested yet.

MAL2Go is built for an upcoming MyAnimeList Unofficial CLI-Based client that is very based (macli). I'm not currently working on macli right now, first I will complete MAL2Go and the YouTube series and then macli will come into development.

If you want macli to release more quickly, please contribute to MAL2Go! Currently, all the methods associated with "manga", "user/manga" packages need to be tested if they are working as intended. Also, the "forums" packages needs to be worked on.

I have a YouTube series on MAL2Go, it only has an episode 0 explaining how to generate a Client ID, Auth Token to work with MAL2Go. I have filmed other videos but they are yet to be released.

All the documentation and source code for MAL2Go can be found at https://github.com/MikunoNaka/MAL2Go
